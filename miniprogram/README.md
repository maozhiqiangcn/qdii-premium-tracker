## 正式发布

微信云托管默认公网域名通常适合调试。正式上线建议：

1. 在云托管服务里绑定自定义域名。
2. 在微信公众平台的小程序后台，把该域名加入 request 合法域名。
3. 把 `config.js` 里的 `API_BASE` 更新为正式域名。

数据仅用于个人跟踪参考，不构成投资建议。
