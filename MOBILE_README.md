# 手机版 LOF 溢价监控

个人主体小程序不适合发布金融行情类工具，所以当前推荐使用手机网页/PWA：

- 本地预览：打开 `http://127.0.0.1:8766/mobile.html`
- 手机访问：把页面部署到静态托管后，用手机浏览器打开并添加到主屏幕
- 实时刷新：页面打开时自动每 60 秒刷新一次
- 前台提醒：溢价率达到阈值时，页面会震动并弹出通知
- 数据失败：显示六小时内的上次成功快照，并明确标注 `缓存数据`

## 接口配置

默认本地预览走同源 `/api/haoetf`。部署到静态站点时，可以在 `mobile-config.js` 中填写远程接口：

```js
window.LOF_API_ORIGIN = "https://你的后端域名";
```

当前根目录的 `server.py` 已提供 `/api/haoetf`。

## 后台提醒

浏览器前台提醒只能在页面打开时工作。要做到后台提醒，建议下一步接一个个人通知通道：

- Bark：适合 iPhone
- PushDeer：适合微信/手机推送
- Telegram Bot：适合海外网络环境

做法是让后端定时检查溢价率，超过阈值后推送到你的手机。
